# NeoMorph Credits

## NeoMorph
The ultimate morph plugin for PaperMC.

## Embedded iDisguise Engine
NeoMorph includes an embedded version of **iDisguise** by **Luisa R. Grether**.

- **Original Author:** Luisa R. Grether (Copyright © 2025)
- **Original Source:** https://github.com/LuisaGretworthy/iDisguise
- **Website:** https://www.luisagrether.de/mc/plugins/idisguise/
- **API Documentation:** https://www.luisagrether.de/mc/plugins/idisguise/javadocs/6-0-1/

The iDisguise source code located in `de.luisagrether.idisguise` and `de.luisagrether.util`
packages is the work of Luisa R. Grether. It has been refactored from a standalone plugin into
an embedded engine for use within NeoMorph, but the core disguise logic remains her original work.

All iDisguise source files retain their original package structure and authorship.
